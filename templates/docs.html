System Overview

The AI Traffic Analytics system uses Computer Vision to detect traffic violations and identify vehicles in real-time. It leverages YOLOv8 (You Only Look Once) architectures for high-speed object detection and EasyOCR for optical character recognition on license plates.

ANPR Engine

Automatic Number Plate Recognition. Crops the plate and uses deep learning to extract alphanumeric characters.

Violation Triggers

Monitors for Helmet violations, Triple Riding (3+ on a bike), and Seatbelt usage for cars.

Intelligent Tracking & Deduplication

1

Persistent ID Tracking

The system assigns a unique ID to every vehicle. Even if the vehicle moves across multiple frames, it is recognized as the same entity.

2

OCR Stabilization

By comparing extracted text strings, the system prevents logging the same vehicle twice even if the tracking ID switches due to temporary obstruction.

Performance Tips

  • Use .mp4 containers with H.264 codec for best browser compatibility.
  • Avoid highly blurry footage; OCR accuracy improves significantly with 720p or 1080p resolution.
  • The system dynamically skips non-essential tasks to keep the background processing under 2x the video length.
