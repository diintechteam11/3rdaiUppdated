Overview

The AI Traffic Analytics API provides developers with high-performance video analysis tools. Process videos asynchronously or connect directly to live RTSP camera streams to detect license plates, helmets, triple riding, and seatbelt violations in real-time.

Detection Triggers

Number Plate Detection

Detects license plates, extracts text, and identifies vehicle color.

Helmet Detection

Identifies two-wheeler riders without helmets.

Triple Riding Detection

Detects more than two people on a single two-wheeler.

Seatbelt Detection

Checks if driver/passenger are wearing seatbelts.

POST

/upload-video

Submit a video for asynchronous processing. The API will return a task_id which you can use to poll for status and results.

Request Body

video_file File (required)

Multipart/form-data video file (mp4, avi, mov, mkv).

triggers String (required)

Comma-separated list of trigger names (from table above).

Code Example (Python)

requests_example.py Python 3.x 
import requests

url = "http://localhost:8000/upload-video"
headers = {"X-API-Key": "your_api_key_here"}
payload = {"triggers": "Number Plate Detection,Helmet Detection"}
files = [("video_file", open("path/to/video.mp4", "rb"))]

response = requests.post(url, headers=headers, data=payload, files=files)
print(response.json())
GET

/video-result/{task_id}

Fetch the current status, processed video URL, and detailed event logs for a specific task.

Response Structure

{
  "task_id": "8a7...",
  "status": "completed", 
  "video_url": "/static/outputs/...",
  "logs": [
    {
      "timestamp": 12.5,
      "event": "Detection (ID: 4)",
      "plate_number": "MH12XY4567",
      "vehicle_color": "Blue"
    }
  ]
}

Status Values

  • queued: Task is waiting for a worker
  • processing: AI is analyzing the video
  • completed: Results are ready
  • failed: Task crashed (check logs)
POST

/connect-camera

Connect to a live RTSP stream for real-time analysis. The connection process is non-blocking.

Request Data (Form-Data)
name
Human-readable name for the camera.
link
RTSP/RTMP stream URL.
triggers
Comma-separated detection types.
GET

/camera-stream/{camera_id}

Returns a standard MJPEG multipart stream. You can embed this directly into an <img> tag's src attribute for visual output.

<img src="http://localhost:8000/camera-stream/your_camera_id" />
